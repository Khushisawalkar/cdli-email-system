# MailFlow: Intelligent Email Delivery System

MailFlow is a prototype email processing system designed to simulate a reliable and user-friendly email delivery pipeline.

## 🚀 Features

- Compose and manage emails through a graphical interface
- Queue-based email processing
- Status tracking (Pending, Sent)
- Dashboard showing system statistics
- Dual-mode operation:
  - Demo Mode (all emails succeed)
  - Simulation Mode (realistic behavior)

## 🧠 Key Idea

Traditional systems often fail silently or block user actions during email delivery.  
MailFlow demonstrates a more reliable approach by separating email handling into a structured processing system.

## 🎯 Purpose

This project is built as a prototype for scalable email infrastructure systems, focusing on:

- Reliability
- User experience
- System transparency

## 🛠 Tech Stack

- Java (Core Logic)
- Java Swing (GUI)

## 📌 Future Improvements

- Retry mechanism with exponential backoff
- Email analytics dashboard
- Integration with SMTP servers
- Background job queue (Redis)
